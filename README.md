# Exchange-Value

## An Exchange Rate Single page App with just Vanilla Javascript, Node.js and HandleBars

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash

nodemon
```

### Developement Server

```bash

localhost:3000
```
